# csvreader

Программа для обработки файлов в формате CSV

## Сборка

```
make
```

## Запуск

```
./csvreader file.csv
```

## Формат входного файла
Первая строка - названия столбцов. Первый столбец всегда пустой. Строки нумеруются целыми положительными числами (необязательно в порядке возрастания). В ячейках хранятся либо целый числа, либо выражения вида \<ARG1 OP ARG2\>, где \<OP\> - операция из списка <+,-,/*>.

## Пример

Входной файл:

```
,A,B,Cell
1,1,0,1
2,2,=A1+Cell30,0
30,0,=B1+A1,5
```

Вывод:

```
,A,B,Cell
1,1,0,1
2,2,6,0
30,0,1,5
```

## Обработка ошибок

Ошибки в ячейке (деление на ноль, циклическая ссылка, несуществующий адрес) выводятся как `#ERROR`. Ячейки, зависящие от ячейки с ошибкой, тоже получают `#ERROR`.

Ошибки формата файла (неверный заголовок, неверное число полей, дублирующийся номер строки) выводятся в stderr, программа завершается с кодом 1.
